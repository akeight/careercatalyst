# Interview Prep evaluation fixtures

Manual scorecard inputs for repeatable review. Do not store API keys.

Suggested cases:

1. Large public software company — Software Engineering
2. Small startup — sparse public info
3. Similarly named companies
4. Frontend position
5. Mobile iOS position
6. Mobile Android position
7. Cross-platform mobile
8. UX / product design
9. Product management
10. Outdated job listing
11. Company with recent major news
12. Conflicting external reports

Score: company resolution, source authority, relevance, freshness, accuracy, citation validity, role lens, uncertainty, questions, no invented experience, usefulness, unsupported-claim rate.
